# OrderOps
OrderOps — SaaS de gestión de pedidos y catálogo para negocios

Sistema de pedidos online para negocios (pastelerías, comida, etc).

## Qué hace

- Catálogo online por negocio
- Carrito + checkout
- Pedidos estructurados (no más WhatsApp caótico)
- Dashboard de gestión
- Confirmación por WhatsApp
- Multi-negocio (SaaS ready)
